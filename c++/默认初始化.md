## 默认初始化

https://en.cppreference.com/w/cpp/language/default_initialization

```c++
class CF{
public:
    int m_b; //不确定值
    int m_a; //不确定值


    explicit CF(int a):m_a(a),m_b(m_a){
        cout<<m_a<<endl;
        cout<<m_b<<endl;
    }
    CF(bool a) {
        m_a = 1;
        cout<<m_a<<endl;
        cout<<m_b<<endl;
    }
    
};

int main() {
    CF d(true);
    return 0;
}
```

m_a和m_b都是不确定值，m_b由于没有被构造函数初始化就会默认初始化，即野值
